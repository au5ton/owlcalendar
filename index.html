How-to

This tool provides an ics feed for the Overwatch League and contenders calendars. It takes the data feeds Blizzard provide from their API and converts it to a format that you can use applications such a Google Calendar to embed a calendar in to. Parameters can be added to limit or format the data as you want them to be displayed.

Example format

To get all calendar data in the basic format, use /calendar

For the calendar with detailed team names and showing scores/spoilers, use /calendar?format=detailed&scores=true

Limiting to teams

To limit the data to only matches that are contested by one or more teams, add the teams parameter, with the three-letter abbreviation separated by a comma.

eg, /calendar?teams=LAV,NYE,BOS will give a feed featuring only games in which Los Angeles Valiant, New York Excelsior, and Boston Uprising are participating in, plus all finals in which the competitors are unknown.

Valid values are:

  • BOS - Boston Uprising
  • GLA - Los-Angeles Gladiators
  • VAL - Los Angeles Valiant
  • FLA/FM - Florida Mayhem
  • SFS - San Francisco Shock
  • SEO - Seoul Dynasty
  • SHD - Shanghai Dragons
  • LDN/LS - London Spitfire
  • DAL - Dallas Fuel
  • HOU/HO - Houston Outlaws
  • NYE - New York Excelsior
  • PHI/PF - Philadeliphia Fusion

Limiting to leagues

You can retrieve multiple league calendars in a single ICS feed using the leagues parameter. If omitted, only Overwatch League will be returned. Valid values are:

  • OWL - Overwatch League. This acronym will also be the current season.
  • OWC - Overwatch World Cup 2018
  • CS2 - Overwatch Contenders - Season 2
  • CS1 - Overwatch Contenders - Season 1

For example, for a feed with detailed info, showing scores, for all Overwatch Contenders Season 1 and 2 matches, use /calendar?format=detailed&scores=true&leagues=CS1,CS2

Say you wanted a calendar with only games featuring the Sydney Drop Bears in Contenders Season 2, you could use /calendar?scores=true&leagues=CS2&teams=SDB

When more data becomes available, functionality will be added for World Cup games. This league will likely be OWC2018.

Limiting to regions

Contenders calendars can also be limited to regions using the regions parameter. Valid regions are:

  • OC/AUS - Australia
  • CN - China
  • EU - Europe
  • KR - Korea
  • NA - North America
  • AS/PAC - Pacific
  • SA - South America

For example, to return all matches in Contenders seasons 1 and 2 for Korea and China only, you could use /calendar?scores=true&leagues=CS1,CS2&regions=KR,CN

Match text format

The default short format is OWL LAV v SHD. Adding the value format=detailed will show extended detail for the calendar event item, and the output format will become Stage 4 - Los Angeles Valiant vs Shanghai Dragons

eg, /calendar?teams=SHD,LAV&format=detailed

Match scores

You can also choose to display match scores for matches which have concluded. To do this, add the parameter scores=show. For example, /calendar?teams=BOS,FLA,LDN&scores=show. This will show in either short or detailed format, and will also change the display to show Winner (3) d Loser (1) instead or Team v Team.

Source code

Source code for those utility is available at https://bitbucket.org/tjsrowe/owlcalendar/.

Build instructions

To build and run, first check out the code. Then, run:
npm install to get all the modules required to run the server
node app.js to run the application server.

That's it.
