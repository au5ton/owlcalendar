Overwatch League ICS calendar

How-to

This tool provides an ics feed for the Overwatch League calendar. It takes the data feed Blizzard provide from their API and converts it to a format that you can use applications such a Google Calendar to embed a calendar in to. Parameters can be added to limit or format the data as you want them to be displayed.

Example format

To get all calendar data in the basic format, use /calendar

Limiting to teams

To limit the data to only matches that are contested by one or more teams, add the teams parameter, with the three-letter abbreviation separated by a comma.

eg, /calendar?teams=LAV,NYE,BOS will give a feed featuring only games in which Los Angeles Valiant, New York Excelsior, and Boston Uprising are participating in, plus all finals in which the competitors are unknown.

Valid values are:

  • BOS - Boston Uprising
  • GLA - Los-Angeles Gladiators
  • VAL - Los Angeles Valiant
  • FLA/FM - Florida Mayhem
  • SFS - San Francisco Shock
  • SEO - Seoul Dynasty
  • SHD - Shanghai Dragons
  • LDN/LS - London Spitfire
  • DAL - Dallas Fuel
  • HOU/HO - Houston Outlaws
  • NYE - New York Excelsior
  • PHI/PF - Philadeliphia Fusion
